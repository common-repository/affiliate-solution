Affiliate Solution is usefull for generating affiliate tracking links of various affiliate channels.

== Description ==

Affiliate Solution is helpful for blogger and coupons websites. It integrates tracking links, so that blogger and coupons website can get commission from store through affiliate channels.

Features
========
- Coupons Management
- Deals Management
- Offers Categories
- Affiliate Channels
- Short Codes 
- SEO Ready
- Automatic Link Generator
- Clicks Recording
- Additional Cashback

== Installation ==

In most cases you can install automatically from WordPress.

However, if you install this manually, follow these steps:

1. Upload the entire `affiliate-solution.zip` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Do I need to add tracking links for every offer?  =

No, It will automatically concatinate tracking links.


If you have any further questions, please submit them [to the support forum](https://wordpress.org/support/plugin/affiliate-solution).
